# battlearena2015

Repo du code Java de l'IA

## Utilisation

### Ligne de commande

* Compiler l'engine
* Editer les variables de la classe `CommandLine.java` afin qu'elles contiennent vos informations
* Lancer le main dans la classe `CommandLine.java`
* Enjoy

### Application Web

* Compiler la webapp
* Deployer sur un tomcat
* Enjoy
